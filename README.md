# Goods Investimentos

Aplicativo para gerenciamento de investimentos incluindo ações brasileiras, criptomoedas, renda fixa, fundos imobiliários brasileiros e ETFs internacionais.

## Tecnologias

- Java 21
- Spring Boot 3.4.2
- TimescaleDB
- Flyway
- TestContainers
- Docker
